# Student Performance Predictor

A Machine Learning project that predicts student final exam marks based on academic factors such as attendance, study hours, internal marks, and assignment scores.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Features
- Linear Regression model
- Random Forest model
- Model evaluation using MSE and R²
- Correlation heatmap
- Prediction visualization
- Interactive web interface using Streamlit

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the model:

python models/train_model.py

Run the web app:

streamlit run app.py
